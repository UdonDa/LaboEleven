# 要件定義

## 購入者
1. LineBotに対して商品番号を送る（ex: ```購入 1```)
2. Botから支払いするように命じられる
3. oneClickで購入

### メリット
購入者にとっては, 研究室にいながら, ものが買える
### デメリット
割高

## 登録者
1. LineBotに対して商品番号を送る（e: ```登録 1```）
2. Botから金額が支払われる
3. 受け取る

### メリット
どこか行くついでに物を買うと, ```テーブル価格 - 買った金額 - 10```のお金が貰える
### デメリット
めんどくさい
